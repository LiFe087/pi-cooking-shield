# 🛡️ PI-Cooking-Shield
*"Cooking up cybersecurity, one threat at a time"*

## 🎯 Overview
Distributed cybersecurity system for SMEs using AI and edge computing.

## 🏗️ Architecture
- **Frontend**: React + TypeScript + Tailwind CSS
- **Backend**: Python Flask microservices  
- **ML Engine**: Distributed inference on Jetson Nano
- **Hardware**: Raspberry Pi 4 + Jetson Nano + FortiGate 100D
